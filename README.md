# WalkFlow-Mac

macOS AppKit app for camera-based gesture control.

## Local Development

For stable local Accessibility/TCC identity without a paid Apple Developer
account, create a local development signing identity once:

```bash
./script/setup_local_signing.sh
./script/build_and_run.sh --verify
```

See [docs/LOCAL_SIGNING.md](docs/LOCAL_SIGNING.md) for the full local signing
setup and troubleshooting notes.
